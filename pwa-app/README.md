1. 解压缩下载的 zip 文件。
2. 运行 npm install 以安装运行服务器所需的依赖项。
3. 编辑 server.js 并设置 DarkSky API 密钥。
4. 运行 node server.js 以在端口 8000 上启动服务器.
5. 打开浏览器选项卡并转到 `http://localhost:8000`

```sh
# install
$ yarn install
# OR
$ npm install

$ npm start
# OR
$ node server.js

# dev
$ ./dev.sh
$ npm run dev

```
